# flowmada-skills

Personal iOS/Swift development and testing skills for Claude Code.

## Available Skills

Skills are published here from the source repository. Use the publish-skill to add or update skills.

## Installation

On your machine:

```bash
/plugin marketplace add flowmada/claude-skills-marketplace
/plugin install flowmada-skills@claude-skills-marketplace --scope user
```
